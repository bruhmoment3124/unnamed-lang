# unnamed-lang

Below is an informal description of the unnamed programming language:

A program in unnamed is a sequence of zero or more statements. Statements change the state of the program.

There are five different types of statements. They are described below:

**Declare**

The keyword **declare**, followed by an identifier, followed by a **;**

**Assignment**

An identifier, followed by **=**, followed by an expression, followed by **;**

**If**

The keyword **if**, followed by **[**, followed by an expression, followed by **]**, followed by a sequence of zero or more statements, followed by the keyword **end**

**While**

The keyword **while**, followed by **[**, followed by an expression, followed by **]**, followed by a sequence of zero or more statements, followed by the keyword **end**

**Print**

The keyword **print**, followed by an identifier, followed by a **;**
